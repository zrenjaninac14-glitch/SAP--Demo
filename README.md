# SAP - School Assistant for Parents demo

Demo prototip jedne Next.js stranice za validaciju ideje sa roditeljima.

## Pokretanje

Potrebno je da Node.js bude instaliran na računaru.

```bash
npm install
npm run dev
```

Zatim otvorite:

```text
http://localhost:3000
```

## Šta demo sadrži

- Hero sekciju sa kratkim objašnjenjem proizvoda.
- Polje za unos teme ili problema deteta.
- Hardkodovanu proveru znanja sa 5 pitanja iz tablice množenja.
- Automatski rezultat sa procentom uspešnosti i kratkom procenom.
- Preporuku sa oblasti za fokus, planom rada i dodatnim zadacima.
- Kontakt sekciju koja prikazuje uspešnu poruku bez slanja podataka.
